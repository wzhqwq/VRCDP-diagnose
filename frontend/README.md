# VRCDP Diagnose Frontend

Svelte 5 + TypeScript frontend for the VRCDP diagnostics workspace.

## Scripts

- `npm run dev` starts the Vite development server.
- `npm run build` creates a production build.
- `npm run preview` serves the production build locally.
- `npm run check` runs Svelte and TypeScript checks.

## Direction

The first UI phase should stay operational and dense: sessions, timelines,
request details, chunk events, pacing windows, markers, and glitches.
